## 📖 About

OneFit Clothing is a modern, fully responsive frontend website designed for an apparel brand. It features product layouts, navigation menus, and interactive UI elements for a smooth browsing experience across all devices.

**Live Demo:** [View Website](https://onefitclothing.netlify.app/)

---

## ✨ Features

- 📱 Fully responsive design (mobile → desktop)
- 🎨 Modern dark theme with neon accents
- 🛍️ Product grid showcase
- 🍔 Mobile hamburger menu
- ✨ Smooth hover animations
- 📧 Newsletter signup
- 🔼 Back to top button

## 🚀 Getting Started

## 🐛 Known Issues

| Issue | Solution |
|-------|----------|
| Images not loading | Check image paths are correct |
| Mobile menu not working | Clear browser cache |
| Animations lag | Reduce effects on older devices |
